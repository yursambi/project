Установил в Windows SSH-клиент PuTTY, подключился к гостевой системе
(Ubuntu 20.04 desktop) с помощью IP-адреса, предварительно установив в ней 
OpenSSH Server и файловый менеджер Midnight Commander.

1. Параметры команды cd:
   ~     -позволяет быстро переместиться в домашний каталог;
   ..    -позволяет перейти на уровень выше;
   ../.. -позволяет перейти на два уровня выше;
   /     -позволяет быстро переместиться в корневой каталог;
   -     -позволяет быстро переместиться  в предыдущий посещённый каталог; 

2. Создал несколько текстовых файлов (с наполнением) с использованием:
   а)команды echo;
   б)редактора vim;
   в)редактора nano;
   г)редактора emacs (тот ещё геморрой :-))

 touch - создать пустой файл;
 echo  - записать в файл с помощью параметра >;
 cat   - просмотреть содержимое файла;
 mkdir - создать директорию (mkdir -p  -создать каталог с подкаталогом); 
 cp    - копировать файл или директорию (параметр -r);
 mv    - переместить файл или каталог (или переименовать);
 rm    - удалить файл (rm -rf -удалить директорию со всем содержимым);

3. Настроил SSH авторизацию по ключам с использованием Putty и PuTTYgen.
